# 行阶梯型矩阵
给出一个矩阵，将其转化为行阶梯型矩阵。是不是很简单！没错！真难！</br>
温馨提示：本代码遵循C99编译标准，我们推荐您使用命令提示符编译本代码。</br>
cd [代码所在目录]</br>
gcc -std=c99 Row-Echelon-Form.c -o Row-Echelon-Form</br>
